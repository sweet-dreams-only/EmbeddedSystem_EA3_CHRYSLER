DaVinci Configurator Pro 4.0 SP6

New features

  • Support AUTOSAR 3.2.1 specification
  • ReadOnly mode for GENy components (with each project handling current state of 'lock' itself)
  • Recorder for ECUC changes
  • Create a stripped down package with information for support requests
  • Support sorting according to SymbolicNameValue in GCE view
  • Change order of containers in GCE view with MoveUp and MoveDown option
  • Support extended filter functionality in all combobox columns of tables
  • 1st Level Validation should be executed completely and independent of existance of Validation.xml
  • Improved performance for loading and saving large ECUC files
  • Improved performance for applying PreConfiguration and Recommended Configuration
  • Improved performance for opening/switching view with extensive Actions
  • New option '$(Package)' for external tools
  • Allow saving a 'purged' configuration of modules (= without PreConfiguration) into ECUC file
  • Provide 'Find Entitiy ...' as toolbar icon of project explorer for quick access
  • Provide quick access to module preferences in context menu of architecture
  • Harmonisation of ECU contents (for float values and empty elements) with GENy and DaVinci Developer
  • Results of 'Detect Unresolved References' now support Jump-To-Error feature

MICROSAR Generator

  • Omit printing the same (Validation) message too often
  • Use only required modules for conversion of ECUC file to different AUTOSAR revision, not complete ECUC file

Fixed issues

  • F6 key support for opening special dialogs does not always work
  • Architecture 'Compare To ...' dialog mixes 'shown' and 'to-be-saved' values of parameters
  • Package license does not work if UNC-notation is used for path definition
  • Unexpected windows sound on field editing
  • XCopy Installation raises wrong 'No controller information available in ...' error message
  • Correct license activation is not recognized in rare cases
  • Starting with dpa-file for invalid package license results in access violation
  • MODULE-DESCRIPTION-REF is not written into ECUC file
  • F5 key creates new values for some parameters and references
  • XSL-Generation does not work for splitted ECUC
  • Validation with external tools does not work for splitted ECUC
  • Merged ECUC file for external tools uses wrong AUTOSAR revision
  • Scrollbars are not shown if not all input fields fit on view of module
  • Default value is shown in View but will not be saved
  • Reference targets will not be found in Group which uses SUB-PACKAGES in module definition
  • Labels are missing sometimes when using Windows 7
  • 'Not empty' check will not be executed correctly for Identifiers of new Container
  • Some Containers are not selectable in GCE view
  • Access violation may occur when deleting UserBlocks in NvM
  • ComSignals can be added while in ReadOnly mode
  • Context menu does not show correct types of Sub-Containers in GCE view
  • After Validation ParameterHelp does no longer work correctly
  • Splitter above Table is not shown correctly
  • Architecture is empty when closing and reopening it
  • PrettyPrint of ECUC file invalidates empty VALUE-REF nodes
  • Minimum relation of Containers is not considered correctly when creating new project
  • DaVinci Project Assistant does not get ECUC-lock when started from 'File/Project Assistant/Project Settings...'
  • PreConfiguration and Recommended Configuration for reference values is not applied correctly
  • Recommended Configuration and Recommended Class Configuration are not applied correctly for ConfigurationSets
  • Commmandline Generation does not accept different camelcase spelling of module
  • Creating 'Dem Events' in 3rd party Dem fails
  • Generation of WdgIf fails for values of WdgExt
  • Cascaded Preconfigured Class Configuration and Recommended Class Configuration are not applied correctly
  • Automatic update of Reference values does not work correctly in some cases
  • Values of Instance-References are not loaded
  • 'Del' key is working in locked GCE view

DaVinci Configurator Pro 4.0 SP5

New features

  • Support of multi selection with bulk change in GCE and tables
  • AutoComplete and extended search functionality for dropdown lists
  • Enhanced usability in dialog for editing references
  • New validation and generation dialog
  • New 'Compare To...' feature in architecture view with possibility to create a simple report for all changed entity values
  • Introducing new naming 'Code Templates' for user changeable files
  • New project on welcome screen now starts DaVinci Project Assistant
  • The sort order of items in GCE view can be configured
  • The parameter information dialog shows the LONG-NAME of a selected entity
  • Support of simple html tags (<b>, <i>, <u>) in control tooltip
  • Integer values can be displayed in binary format (optional as replacement to dec/hex display)
  • Support of unsigned 64-Bit integer values
  • Create a report which contains all changed entity values
  • Parse package license folder automatically (without usage of System.xml)
  • Improved handling of large Windows fonts (120 dpi)
  • Recalculate derived parameters after opening ECUC file

  • Windows 7 support

  • Internal name of module supports underscores (Note: If internal module name contains underscores, Identifier-notation may not be used for this module)
  • View Tab names of instanceable modules are simplified

Fixed issues

  • ShortName of module configuration is fixed if PreConfiguration is used
  • 'Save' is disabled although module configuration was changed during loading
  • Sorted Table shows wrong value on related Panels
  • Handling of instancable modules
  • 'Recent Projects' list is not shown at correct position on japanese Windows XP
  • References were not updated correctly if target module was renamed
  • 'Revert To Initial Values' in Architecture can sometimes result in access violation
  • Access to not existing optional parameter results in index out of bounds exception
  • 'Extended' xsd-check (= ShortNames may have up to 128 chars) reports errors for long reference values
  • 'Undefined' checkboxes are not shown correctly for related Panels of TreeFrames and TableFrames
  • Controls are not enabled correctly on Os-view
  • MinRelation of Containers, Parameters etc is considered before loading a project instead of after loading it
  • Switching pages in comfort view can result in access violation
  • When switching pages in GCE view sometimes certain values are not shown correctly
  • Selected Choice Containers are not loaded correctly if lower multiplicity is set wrong in module definition
  • In some cases multiple configuration container is not shown correctly in TreeFrame
  • Update module configuration when saving ECUC after 'Revert To Initial Values' was used
  • Saving ECUC fails due to unexplained reason
  • Wrong context is used in ActionLists and for enabling Controls

DaVinci Configurator Pro 4.0 SP3

New features

  • Support of Fold Containers in GCE to enhance clarity of GCE tree
  • Support of Cut, Copy and Paste of containers in GCE
  • Create validition log file in XML format where all executed validation rules are signed per ID with status OK or NOK
  • Support use of variable names in folder specifications
  • Extend parameter context menu with new item to undefine a parameter
  • Enhance usability of assigning package license to projects
  • Provide new menu item in Architecture view to revert module configuration to default with optional Recommended Configuration
  • Enhance performance on loading projects and module editors
  • Extend find parameter dialog with possibility to search for parameter comments
  • Extend table context menu to insert or delete selected row
  • Extend project menu with new item to test multiple configuration container names for uniqueness
  • Read and write ECUC files with more than one AR-PACKAGE
  • Support message numbers in all user messages
  • Close nested messages automatically if they only contain uncritical severity messages
  • Harmonize XSD validation with possibility to ignore shortname length violation
  • Extend message context menu with item to open files with unknown file association
  • Support AUTOSAR 3.0 Rev6 and 3.1 Rev4
  • Support bit-operations in formula descriptions
  • Status line reworked for better understandability
  • Consider MinRelation of Containers, Parameters etc when creating new project

MICROSAR Generator

  • Generation.log file format changed to XML
  • Print user notification in module selection dialog for dependencies of modules that become deactivated
  • Support starting generator with dpa-file
  • Provide option to switch off the creating of *.bak files during generation
  • Save user code from Removed Code Area on restart of generation process

Fixed issues

  • Memory leaks removed
  • The parameter help window is empty after insertion of a new row in a table
  • The range of integer or float param is not displayed in the parameter help window
  • Wrong short cut description in context menu for parmeter help found (should be Shift+F1)
  • The project explorer becomes not synchronized in some case
  • Set value to default fails with wrong pasted text in some case
  • Jump-To-Error fails in some cases
  • ECUC change detection shall not be executed if application is not focused
  • Error messages of 3rd party generators are not well-formed if project path contains blank char
  • Button for Hex/Dec format wrongly stays disabled in some case
  • Access violation when changing editors in some case
  • ECUC pretty print becomes not well-formed in some case
  • Path specifications containing '#' cause errors on creating editors
  • Tooltips are sometimes not readable
  • An existing Recommended Configuration is not considered when a module is added to the project
  • Shortname naming algorithm skips free numbers
  • DaVinci Project Assistant does not save ECUC before "update ecuc project..." is executed
  • The error Messages raised from DaVinci Project Assistant are not displayed completely
  • Relative project paths specified by DaVinci Project Assistant are not generated to the correct folder
  • Loading a file can 're-enter' optional Containers
  • Saving fails if 'MSXML 6.0 SP2' is used
  • Performance issue when opening large comfort views
  • Parameters $(EcucFile) and $(EcucFolder) are passed incorrectly to external Validator/Generator if 'Supply AUTOSAR Revision' option is used

DaVinci Configurator Pro 4.0 SP2

New features

  • Display names of module perspectives in project explorer
  • Usage of license.bin becomes obsolete
  • Support PreConfigured and Recommended Class Configuration
  • The display format of Integer parameters can be changed by user to decimal, hex or binary format
  • Support GENy dll for generation of SWC
  • Full integration of new Project Assistent in user interface
    • Create new ECUC project based on system extract
    • Update current ECUC project with new modified system extract
    • Edit common project settings
  • Establish context menu for each parameter to support actions to
    • add or modify a comment string
    • reset the parameter value to the specified default value
    • reset the parameter value to the specified recommended value
    • open a popupwindow with all information raised from definition
  • Support AUTOSAR release 3.1 (together with accrued patch releases 3.0.3, 3.0.4, 3.1.1, 3.1.2)
  • Support usage of identifier.xml to cover extensions of AUTOSAR specification
  • Support derived parameter types with Vector formula description
  • Support concurrent ECUC access between MICROSAR tools

MICROSAR Generator

  • Restrict loading to necessary dependent BSW modules to enhance performance

Fixed issues

  • Missing external generator will not be reported
  • Renaming SymbolicNameRef parameter results in invalid value
  • If an existing but invalid package license entry is found in any project, the project will not be loaded
  • Save module configuration to ECUC only if really changed
  • Deactivating modules does not work in some cases
  • Parsing definition files fails sometimes if additional files found e.g. /bswmd/project.pj
  • ECUC change detection hangup in a special environment
  • The mousepointer will not be set back to default on leaving project explorer
  • Relative paths are resolved not relative to project folder
  • Import Append fails sometimes
  • Context menu item 'Jump To ...' becomes not enabled
  • Calling external generator removes foreign created temp files

DaVinci Configurator Pro 4.0 SP1

New features

  • Provide finding AUTOSAR objects in current configuration
  • Provide new menu item to launch GENy with current project configuration
  • New message register 'Result List' for a message summary grouped by message type
  • Provide new context menu item in architecture for validating module definition file(s)
  • Provide new context menu item in architecture for importing module configuration with replace or append option
  • Provide new context menu item in architecture for opening user added documents
  • Display user notification if current ECUC file is changed from other application (Change Detection)
  • Display dependencies between modules in implementation help window
  • Display AUTOSAR origin, derivation and destination reference in parameter help window
  • Provide new context menu to close selected tab or all tabs except selected one
  • Print warning message on loading a module configuration with a different definition reference except standard definition reference
  • Assign vendor-specific definition on loading a module with a standard AUTOSAR definition
  • Validation of a module can be performed completely and without breaking on first error
  • Provide new menu item to verify UUID values of current project
  • Improvement on calling GENy generator
  • Provide special GCE view for Com- and IPduSignals
  • Provide new .profile setting to launch associated application after creating a report file

MICROSAR Generator

  • Support -tswc command line option to set directory name for generated SWC description files
  • Support -tuser command line option to set the target directory name for generated user-changeable files
  • Support -h command line option for help

DaVinci Difference Viewer

  • Display of semantic differences between AUTOSAR XML files
  • Supports System Description files, SWC Description files and ECU-C files
  • Integrated into Windows Explorer context menu for files with extension .arxml

Fixed issues

  • Extend handling of undefined boolean parameter values
  • 3rd party generator cannot be started if its working directory contains blanks in path specification
  • 3rd party modules are not shown sometimes in project explorer
  • Wrong architecture mapping for FEE module
  • Create tooltip and Ctrl-F1 to access parameter information for locked parameters in GCE mode
  • Redesign of register 'Changes' in message window
  • Different display of messages, no multiple-selection of messages, missing scrollbars
  • Cursor moves to left position in some GCE dialog
  • Problems with 'Do not display the next time' switch in module selection of validation and generation dialog
  • Bad performance on creating GCE perspective for modules with a large count of entities
  • Display issues with different Windows design schemas
  • 'Do not modify ...' comment in generated user-changeable output files
  • Os TimingAnalyzer does not start in some cases
  • 'Jump to ...' to does not work properly
  • MyController setting will not be saved in .profile
  • Wrong 'Cannot map data' warnings are printed
  • Implementation-Info will not be shown correctly for split BSWMD files
  • VisualStudio locks generated file and raises exception on rewriting output
  • Loaded InstanceRefs will be deleted on saving
  • Import Append/Replace does not work properly in some cases
  • SymbolicNameReferences have no value list if destination definition file is missing
  • Exception is raised in spin columns
